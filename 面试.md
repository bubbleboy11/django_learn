答:

# 方法一

11、循环names列表，打印每个元素的索引值和元素，当索引值为偶数时，把对应的元素改成 - 1。（考点列表元素，索引循环）

答:

for index, i in enumerate(names):

    if index % 2 == 0:
        names[index] = -1

        print(index, 1)

print(names)
